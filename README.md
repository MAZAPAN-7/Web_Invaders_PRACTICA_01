# Space Invaders – Arcade Menu Demo

Interfaz web inspirada en los **menús clásicos de recreativas de los años 80**, con una estética neón y ambiente arcade al estilo *salones recreativos* (inspiración Stranger Things).

Este proyecto forma parte de una **práctica de Diseño Web**, centrada en la creación de una **demo navegable** y una **landing page promocional** para un videojuego ficticio.

---

## Descripción del proyecto

El proyecto se divide en dos partes principales:

### 1️ Landing Page
Una página vertical estilo *itch.io* donde:
- El primer elemento visible es el **recuadro del juego con botón Play**
- Al hacer scroll se muestra:
  - Descripción del juego
  - Características
  - Contenido visual
  - Formulario de acceso a beta

La landing actúa como página promocional del juego.

---

### 2️ Demo – Menú principal arcade
Una **demo interactiva** que simula el menú principal de un videojuego arcade.

Incluye:
- Menú principal:
  - New Game
  - Continue
  - Quit
- High Scores con colores por ranking
- Selección de nave
- Selección de mapa
- Menú de opciones:
  - Volumen
  - Dificultad
  - Idioma
- Menú de controles

Todo se gestiona en una sola página (`demo.html`) mediante **paneles y estados**, simulando pantallas de un videojuego real.

---

## Dirección artística

La estética está inspirada en:
- Salones recreativos clásicos
- Máquinas arcade
- Neones, luces de tubo y cartelería retro

Características visuales:
- Paleta neón (amarillo arcade, magenta, cian)
- Texto “INSERT COIN” parpadeante estilo recreativa

---

## Tecnologías utilizadas

- **HTML5**
- **CSS3**
  - Grid Layout
  - Animaciones y efectos visuales
- **JavaScript**
- **jQuery** (gestión de paneles y estados)

No se utilizan frameworks externos para el layout con el objetivo de mantener el proyecto ligero y controlado.
